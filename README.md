# 会议热力图分析工具

一个简单易用的会议时间分析工具，通过热力图直观展示各组织在工作日不同时间段的会议分布情况。

## 功能特点

- 📊 **热力图展示** - 清晰展示周一至周五各时间段的会议密度
- 📁 **文件上传** - 支持 CSV 和 XLSX 格式文件上传
- 🔄 **自动处理** - 自动替换中文冒号（：→:）
- 📈 **统计分析** - 显示总会议数、组织数量、会议日期等统计信息
- 🏢 **组织排行** - 按会议次数展示各组织统计

## 在线使用

访问 GitHub Pages 部署地址即可使用：
https://panegyric0926.github.io/conference-heatmap/

## 文件格式

支持 CSV 或 XLSX 文件，每行格式如下：

```
日期,时间,组织名称
```

### 示例

```csv
20250926,15,对外部
20250929,15,例会
20251013,10:30,人事处
20251014,15,记者部
```

### 字段说明

| 字段 | 格式 | 示例 |
|------|------|------|
| 日期 | yyyymmdd | 20250926 |
| 时间 | 小时 或 小时:分钟 | 15 或 10:30 |
| 组织名称 | 文本 | 人事处 |

## 本地运行

直接在浏览器中打开 `index.html` 文件即可使用。

## 技术说明

- 纯前端实现，无需后端服务
- 使用 [SheetJS](https://sheetjs.com/) 解析 Excel 文件
- 响应式设计，支持移动端访问
